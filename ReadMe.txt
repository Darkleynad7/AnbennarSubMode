Every time u want to work on your mod:

1. Open git bash
2. type: cd Desktop/*the folder on your desktop where u have the mod*
3. press enter
4. type: git pull origin master
5. press enter
6. start working on ur mod


After u finish working on ur mod

1. Open git bash
2. type: cd Desktop/*the folder on your desktop where u have the mod*
3. press enter
4. type: git add .
5. press enter
6. type: git commit -m "A name u want for ur commit"
7. press enter
8. type: git push origin master
9. press enter
